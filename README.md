# Evento Documentation

Official documentation for the Evento Public API, React Embed Components, and MCP Server.

## What's Documented

- **Public API**: RESTful API for accessing event and user data
- **React Embeds**: Drop-in React components for displaying Evento content
- **MCP Server**: Claude Desktop integration for AI-powered event management

## Local Development

This documentation is built with [Mintlify](https://mintlify.com).

```bash
# Install Mintlify CLI
npm i -g mintlify

# Start local dev server
mintlify dev
```
