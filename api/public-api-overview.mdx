## Base URL

```http
https://evento.so/api/public/v1
```

## Authentication

All public endpoints require an API key.

Preferred header:

```http
x-evento-api-key: YOUR_API_KEY
```

Alternative header:

```http
Authorization: Bearer YOUR_API_KEY
```

## Rate limits

Public API keys are limited to `1000` requests per day, per key.
