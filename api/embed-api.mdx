## Overview

Embed API is optimized for browser-first integrations.

<CardGroup cols={2}>
  <Card title="No authentication" icon="unlock">
    Safe for client-side rendering. No API key required.
  </Card>
  <Card title="CORS Enabled" icon="globe">
    Cross-origin access supports `GET` and `OPTIONS`.
  </Card>
  <Card title="Compact payloads" icon="feather-pointed">
    Response objects are tuned for embeds and widgets.
  </Card>
  <Card title="Computed status" icon="clock">
    Status is returned as `upcoming`, `ongoing`, or `past`.
  </Card>
</CardGroup>

## Base URL

```
https://evento.so/api/embed/v1
```

## CORS headers

```http
Access-Control-Allow-Origin: *
Access-Control-Allow-Methods: GET, OPTIONS
Access-Control-Allow-Headers: Content-Type
```
