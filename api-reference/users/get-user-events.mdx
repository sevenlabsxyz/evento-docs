Retrieve all public events created by a specific user on the Evento platform.

<Info>
This endpoint returns all public events **created by** the specified user. To get events the user is attending (RSVP'd), use the profile-events endpoint instead.
</Info>

## Endpoint

```
GET https://api.evento.so/api/public/v1/users/{username}/events
```

## Path Parameters

<ParamField path="username" type="string" required>
  The username of the Evento user
  
  Example: `johndoe`
</ParamField>

## Query Parameters

<ParamField query="since" type="string" format="date-time">
  Filter events starting from this date (inclusive)
  
  Format: ISO 8601 (`YYYY-MM-DDTHH:mm:ssZ`)
  
  Example: `2024-01-01T00:00:00Z`
</ParamField>

<ParamField query="from" type="string" format="date-time">
  Alias for `since` parameter
  
  Format: ISO 8601 (`YYYY-MM-DDTHH:mm:ssZ`)
</ParamField>

<ParamField query="to" type="string" format="date-time">
  Filter events up to this date (inclusive)
  
  Format: ISO 8601 (`YYYY-MM-DDTHH:mm:ssZ`)
  
  Example: `2024-12-31T23:59:59Z`
</ParamField>

## Headers

<ParamField header="x-evento-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Examples

<CodeGroup>

```bash cURL
curl -X GET "https://api.evento.so/api/public/v1/users/johndoe/events?since=2024-01-01T00:00:00Z" \
  -H "x-evento-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.evento.so/api/public/v1/users/johndoe/events?since=2024-01-01T00:00:00Z',
  {
    method: 'GET',
    headers: {
      'x-evento-api-key': 'YOUR_API_KEY'
    }
  }
);

const result = await response.json();

if (result.success) {
  console.log(`Found ${result.data.length} events`);
  result.data.forEach(event => {
    console.log(`- ${event.title} on ${event.start_date}`);
  });
}
```

```python Python
import requests
from datetime import datetime

url = "https://api.evento.so/api/public/v1/users/johndoe/events"
params = {
    "since": "2024-01-01T00:00:00Z",
    "to": "2024-12-31T23:59:59Z"
}
headers = {
    "x-evento-api-key": "YOUR_API_KEY"
}

response = requests.get(url, params=params, headers=headers)
result = response.json()

if result["success"]:
    print(f"Found {len(result['data'])} events")
    for event in result["data"]:
        print(f"- {event['title']} on {event['start_date']}")
```

```php PHP
<?php
$curl = curl_init();

$params = http_build_query([
    'since' => '2024-01-01T00:00:00Z',
    'to' => '2024-12-31T23:59:59Z'
]);

curl_setopt_array($curl, array(
  CURLOPT_URL => 'https://api.evento.so/api/public/v1/users/johndoe/events?' . $params,
  CURLOPT_RETURNTRANSFER => true,
  CURLOPT_HTTPHEADER => array(
    'x-evento-api-key: YOUR_API_KEY'
  ),
));

$response = curl_exec($curl);
curl_close($curl);

$result = json_decode($response, true);

if ($result['success']) {
    echo "Found " . count($result['data']) . " events\n";
    foreach ($result['data'] as $event) {
        echo "- {$event['title']} on {$event['start_date']}\n";
    }
}
?>
```

## Notes

<Note>
- This endpoint returns events **created by** the user, not events they're attending
- Events are sorted by creation date (most recent first)
- Only published, public events are returned
- Date filtering applies to the event's start date
- An empty array is returned if no events match the criteria
</Note>

<Tip>
Use date filtering to reduce the response size and improve performance when dealing with users who have many events.
</Tip>
