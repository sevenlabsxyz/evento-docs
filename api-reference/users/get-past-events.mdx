Retrieve all past public events hosted by a specific user, sorted by most recent first.

<Info>
This endpoint returns only **past events** created by the user, sorted by start date in descending order (most recent first). Perfect for displaying a user's event history.
</Info>

## Endpoint

```
GET https://api.evento.so/api/public/v1/users/{username}/past-events
```

## Path Parameters

<ParamField path="username" type="string" required>
  The username of the Evento user
  
  Example: `johndoe`
</ParamField>

## Query Parameters

<ParamField query="limit" type="integer" min="1" max="100" default="10">
  Number of results to return per page
  
  - Minimum: 1
  - Maximum: 100
  - Default: All past events (no limit)
  
  Example: `10`
</ParamField>

<ParamField query="offset" type="integer" min="0" default="0">
  Number of results to skip for pagination
  
  - Minimum: 0
  - Default: 0
  
  Example: `20` (skip first 20 results)
</ParamField>

## Headers

<ParamField header="x-evento-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Examples

<CodeGroup>

```bash cURL
curl -X GET "https://api.evento.so/api/public/v1/users/johndoe/past-events?limit=5" \
  -H "x-evento-api-key: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.evento.so/api/public/v1/users/johndoe/past-events?limit=10&offset=0',
  {
    method: 'GET',
    headers: {
      'x-evento-api-key': 'YOUR_API_KEY'
    }
  }
);

const result = await response.json();

if (result.success) {
  console.log(`Found ${result.data.length} past events`);
  
  result.data.forEach(event => {
    const date = new Date(event.start_date);
    const daysAgo = Math.floor((Date.now() - date) / (1000 * 60 * 60 * 24));
    console.log(`- ${event.title} (${daysAgo} days ago)`);
  });
}
```

```python Python
import requests
from datetime import datetime, timezone

url = "https://api.evento.so/api/public/v1/users/johndoe/past-events"
params = {
    "limit": 10,
    "offset": 0
}
headers = {
    "x-evento-api-key": "YOUR_API_KEY"
}

response = requests.get(url, params=params, headers=headers)
result = response.json()

if result["success"]:
    print(f"Found {len(result['data'])} past events")
    
    for event in result["data"]:
        start = datetime.fromisoformat(event["start_date"].replace("Z", "+00:00"))
        days_ago = (datetime.now(timezone.utc) - start).days
        print(f"- {event['title']} ({days_ago} days ago)")
```

```php PHP
<?php
$curl = curl_init();

$params = http_build_query([
    'limit' => 10,
    'offset' => 0
]);

curl_setopt_array($curl, array(
  CURLOPT_URL => 'https://api.evento.so/api/public/v1/users/johndoe/past-events?' . $params,
  CURLOPT_RETURNTRANSFER => true,
  CURLOPT_HTTPHEADER => array(
    'x-evento-api-key: YOUR_API_KEY'
  ),
));

$response = curl_exec($curl);
curl_close($curl);

$result = json_decode($response, true);

if ($result['success']) {
    echo "Found " . count($result['data']) . " past events\n";
    
    foreach ($result['data'] as $event) {
        $date = new DateTime($event['start_date']);
        $now = new DateTime();
        $interval = $now->diff($date);
        echo "- {$event['title']} ({$interval->days} days ago)\n";
    }
}
?>
```

## Notes

<Note>
- **Past** means `start_date < current_time`
- Events are sorted by start date in **descending order** (most recent first)
- Only events **created by** the specified user are returned
- Empty array is returned if the user has no past events
- Use pagination parameters for users with extensive event history
</Note>

<Tip>
This endpoint is ideal for:
- Building event history pages
- Showing a user's track record
- Analytics on past event performance
- Creating event portfolios
</Tip>
