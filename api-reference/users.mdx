## Endpoint

```
GET /users/:username/events
```

Retrieves a list of events associated with a user. Supports filtering by event type (upcoming, past, profile) and pagination.

## Path Parameters

<ParamField path="username" type="string" required>
  The user's unique username (without @ symbol)
  
  Example: `satoshi`
</ParamField>

## Query Parameters

<ParamField query="type" type="string" default="upcoming">
  Filter events by type:
  - `upcoming` - Future events only
  - `past` - Past events only
  - `profile` - All events visible on user's profile
</ParamField>

<ParamField query="limit" type="number" default="20">
  Number of events to return per page (max: 100)
</ParamField>

<ParamField query="offset" type="number" default="0">
  Number of events to skip (for pagination)
</ParamField>

## Request Examples

<RequestExample>

```bash cURL - Upcoming Events
curl "https://api.evento.so/public/v1/users/satoshi/events?type=upcoming&limit=10" \
  -H "Authorization: Bearer evento_xxxxxxxxxxxxxxxxxxxxxxxx"
```

```bash cURL - Past Events
curl "https://api.evento.so/public/v1/users/satoshi/events?type=past&limit=20&offset=20" \
  -H "Authorization: Bearer evento_xxxxxxxxxxxxxxxxxxxxxxxx"
```

```javascript JavaScript
const username = 'satoshi';
const params = new URLSearchParams({
  type: 'upcoming',
  limit: '10',
  offset: '0'
});

const response = await fetch(
  `https://api.evento.so/public/v1/users/${username}/events?${params}`,
  {
    headers: {
      'Authorization': `Bearer ${process.env.EVENTO_API_KEY}`
    }
  }
);

const { data } = await response.json();
console.log(data.events);
```

```python Python
import os
import requests

username = 'satoshi'
params = {
    'type': 'upcoming',
    'limit': 10,
    'offset': 0
}
headers = {
    'Authorization': f'Bearer {os.environ["EVENTO_API_KEY"]}'
}

response = requests.get(
    f'https://api.evento.so/public/v1/users/{username}/events',
    params=params,
    headers=headers
)

data = response.json()
print(data['data']['events'])
```

## Pagination Example

To fetch all events for a user, use pagination:

<CodeGroup>

```javascript Pagination Loop
async function getAllUserEvents(username) {
  const allEvents = [];
  let offset = 0;
  const limit = 50;
  let hasMore = true;

  while (hasMore) {
    const params = new URLSearchParams({
      type: 'upcoming',
      limit: limit.toString(),
      offset: offset.toString()
    });

    const response = await fetch(
      `https://api.evento.so/public/v1/users/${username}/events?${params}`,
      {
        headers: {
          'Authorization': `Bearer ${process.env.EVENTO_API_KEY}`
        }
      }
    );

    const { data } = await response.json();
    allEvents.push(...data.events);
    
    hasMore = data.pagination.hasMore;
    offset += limit;
  }

  return allEvents;
}
```

```python Pagination Loop
import os
import requests

def get_all_user_events(username):
    all_events = []
    offset = 0
    limit = 50
    has_more = True
    
    headers = {
        'Authorization': f'Bearer {os.environ["EVENTO_API_KEY"]}'
    }
    
    while has_more:
        params = {
            'type': 'upcoming',
            'limit': limit,
            'offset': offset
        }
        
        response = requests.get(
            f'https://api.evento.so/public/v1/users/{username}/events',
            params=params,
            headers=headers
        )
        
        data = response.json()['data']
        all_events.extend(data['events'])
        
        has_more = data['pagination']['hasMore']
        offset += limit
    
    return all_events
```
